Hive Tools by @hivetrending

Utilities

  • hiveisbeautiful - real-time visualization of Hive blocks contents
  • hiveRSS - create RSS feeds for any Hive blog
  • randomhive - show a random recently-published Hive blog post
  • supply - show current Hive and HBD token supply info

Streams

  • accountstream - scrolling stream view of Hive account creation
  • commentstream - scrolling stream view of Hive comments
  • convertstream - scrolling stream view of Hive collateralized conversions
  • firehive - firehose stream of all hive activities
  • hbdstream - scrolling stream view of HBD transfers
  • transferstream - scrolling stream view of Hive and HiveEngine transfers

Monitors

  • DLUX Monitor - show current DLUX side-chain node list and other info
  • DUAT Monitor - show current DUAT (Ragnarok claim chain) node list and other info
  • SPKCC Monitor - show current SPKCC (Spk Network Claim Chain) node list and other info
  • SPK Testnet Monitor - show current Spk Testnet (Spk Network Test Net) node list and other info
